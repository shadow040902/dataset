Overview

About the Dataset

The STVD-FC dataset is related to the Fact-checking problem. It focuses on French political discourses and encompasses the 2022 French presidential election, covering the period from February 1 up to May 1, 2022. The constitution process is detailed in [1].

It contains about 1,330 fact-checked claims scraped from the fact-checking service Factoscope. For the video counterpart, nearly 6,730 TV programs representing a total of 6,540 hours were captured with a TV workstation, alongside metadata.

Dataset Structure

The dataset is organized as follows:

  • →A root directory per part, with subdirectories grouping daily/weekly TV program instances
  • →Audio/video files encoded in MPEG-4 format
  • →TV metadata files in XMLTV format
  • →A global XML file with fact-checked claims and an XML schema

Naming Convention

PartX
Directory for a part of the dataset where X ∈ {1, …, 8}
Hashcode
Subdirectory name in hexadecimal form (160-bit / 40 characters), e.g. 0a83...6b1e
ts
Timestamp for subdirectory: YEAR MONTH DAY_HOURS MINUTES SECONDS, e.g. 20220211_105000

Hello World Index

Baseline NLP & CV Index

For kick-off purposes, STVD-FC is provided with a "hello world" index giving baseline results of NLP and CV methods. It follows the same naming convention as the root dataset: \PartX\Hashcode\ts\

↓ index.zip (~216 MB)   ⊞ keywords.json

ts_transcript
Audio transcription in JSON format using OpenAI Whisper (whisper-tiny model, ~85% accuracy). Includes lemmas and named entities via Spacy. 
{
  "file": "ts_audio",
  "text": "...",
  "lemmas": ["...","..."],
  "named_entities": ["...","..."]
}
ts_spot
Keyword spotting in JSON format using FuzzyWuzzy with ~415 reference keywords at 90% matching threshold. 
{
  "file": "ts_audio",
  "keywords": [
    [id, "kwref", pos, d, "kwspot"],
    ...
  ]
}
